# Jewelry Image Puller (Windows Local Version)

โปรแกรมสำหรับดึงรูปภาพสินค้าจากไดรฟ์ในเครื่อง (Local Drive/Network Drive) ตามรหัสสินค้า

## วิธีใช้งาน
1. **Source Drive:** เลือกโฟลเดอร์หลักที่มีหมวดหมู่สินค้าอยู่ (โฟลเดอร์ที่มี Ring, Earring, Necklace, etc.)
2. **Destination Folder:** เลือกโฟลเดอร์ที่คุณต้องการให้โปรแกรม copy รูปไปวาง
3. **Product IDs:** วางรหัสสินค้าที่ต้องการดึงรูป (ใส่ได้หลายรายการ แยกบรรทัดกัน)
4. กดปุ่ม **🚀 START COPYING**

## โครงสร้างที่โปรแกรมค้นหา
โปรแกรมจะวิ่งไปที่:
`Source Drive` > `{Category}` > `{Category} {Range}` > `รูปสินค้า` > `{ProductID}.jpg`

ตัวอย่าง: `R-10250`
-> ค้นหาในโฟลเดอร์ `Ring`
-> เข้าไปที่โฟลเดอร์ `Ring 10201-10400`
-> เข้าไปที่โฟลเดอร์ `รูปสินค้า`
-> Copy ทุกไฟล์ที่มีคำว่า `R-10250` ออกมา

## หมายเหตุ
- โปรแกรมรองรับการค้นหาแบบไม่สนตัวพิมพ์เล็ก-ใหญ่ (Case-insensitive)
- หากไม่เจอโฟลเดอร์ `รูปสินค้า` โปรแกรมจะค้นหาในโฟลเดอร์ช่วงรหัสโดยตรงแทน
- รหัสสินค้าที่ใส่ต้องมีรูปแบบ `{อักษร}-{ตัวเลข}` เช่น `R-10250`
