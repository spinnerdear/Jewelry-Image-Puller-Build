# Jewelry Image Puller (Pro Edition)

โปรแกรมอัจฉริยะสำหรับดึงรูปภาพสินค้าจากไดรฟ์ในเครื่อง (Local/Network Drive) โดยค้นหาตามรหัสสินค้า (Product ID) รองรับทั้งรหัสรูปแบบใหม่และเก่า พร้อมระบบเลือกรูปภาพผ่านหน้าจอ

## 🚀 คุณสมบัติหลัก (v1.7)
*   **Dual-Code Search:** ค้นหาพร้อมกันทั้งรหัสใหม่ (เช่น `R-10501`) และรหัสเก่า (เช่น `TRK-10501`) อัตโนมัติ
*   **Smart Excel Import:** นำเข้ารหัสสินค้าจากไฟล์ Excel ได้โดยตรง พร้อมระบบเลือกคอลัมน์และรองรับ Merged Cells
*   **Visual Photo Selection:** หากเจอรูปที่รหัสใกล้เคียงกัน (คนละรหัสลูกค้าหรือคนละเวอร์ชัน) จะมีหน้าต่างแกลเลอรี่ให้เลือกรูปที่ต้องการจริงก่อน Copy
*   **Performance Turbo:** ระบบค้นหาแบบ Scoped Deep Search (หาเฉพาะหมวดหมู่ที่เกี่ยวข้อง) และระบบ Cache รายชื่อไฟล์ เพื่อความรวดเร็วสูงสุดและป้องกันโปรแกรมค้าง
*   **ID Normalization:** แก้ไขรหัสที่พิมพ์ผิดให้อัตโนมัติ (เช่น ลืมขีดกลาง หรือมีช่องว่าง)
*   **Overwrite Protection:** ป้องกันการเขียนทับไฟล์เดิม โดยการเติม `_NEW` ต่อท้ายชื่อไฟล์ให้อัตโนมัติ

## 📂 โครงสร้างการค้นหา
โปรแกรมจะค้นหาตามลำดับความสำคัญดังนี้:
1.  `Source Drive` > `{Category}` (เช่น Ring, Earring)
2.  `...` > `{Category} {Range}` (เช่น Ring 10401-10600)
3.  `...` > `รูปสินค้า` (ถ้ามี)
4.  **Scoped Deep Search:** สแกนหาทุกโฟลเดอร์ภายใต้หมวดหมู่นั้นๆ หากหาในตำแหน่งปกติไม่เจอ

## 🛠 ความต้องการของระบบ (สำหรับรันผ่านสคริปต์ .py)
*   Python 3.10+
*   Libraries: `pandas`, `openpyxl`, `xlrd`, `pillow`
    ```bash
    pip install pandas openpyxl xlrd pillow
    ```

## 📖 วิธีใช้งาน
1.  **เลือก Source Drive:** โฟลเดอร์หลักที่มีหมวดหมู่สินค้าอยู่
2.  **เลือก Destination:** โฟลเดอร์ปลายทางที่ต้องการเก็บรูป
3.  **ใส่รหัสสินค้า:**
    *   วางรหัสลงในช่อง (รองรับ Ctrl+V และคลิกขวา)
    *   หรือกด **📊 IMPORT EXCEL** เพื่อดึงรหัสจากไฟล์งาน
4.  กดปุ่ม **🚀 START COPYING IMAGES**
5.  หากมีหน้าต่างรูปภาพเด้งขึ้นมา ให้เลือกรูปที่ต้องการแล้วกดตกลง
6.  ตรวจสอบรายงานรหัสที่หาไม่เจอได้ที่ไฟล์ `missing_report_xxx.txt` ในโฟลเดอร์ปลายทาง
